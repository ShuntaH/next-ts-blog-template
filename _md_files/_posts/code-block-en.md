# Code syntax highlighting is applied to code blocks.
## JavaScript

```js
const hoge = 'hoge';
console.log(hoge);
```

## HTML
Check that the img tag written in the code block is displayed as a string, not an image.
```html
<div class="hoge">
  <p>fuga</p>
    <img src="/hoge" alt="hoge">
</div>
```

## CSS

```css
.hoge {
color: #fff;
background-color: #000;
}
```

## Markdown

```md
# hoge
## fuga
### piyo
```

## Python

```py
hoge = 'hoge'
print(hoge)
```
